## 🚀 HỆ THỐNG QUẢN LÝ THUÊ XE MÁY (TPS PROJECT)
## 📌 Giới thiệu

- Đây là dự án Hệ thống quản lý thuê xe máy được xây dựng cho dự án cuối kỳ môn học Hệ thống thông tin quản lý (MIS).

- Dự án mô phỏng hoạt động của một doanh nghiệp cho thuê xe máy và tập trung vào việc triển khai Hệ thống xử lý giao dịch (Transaction Processing System – TPS) nhằm xử lý các nghiệp vụ phát sinh hằng ngày như đặt xe, tính tiền, lưu lịch sử thuê,...

**Thông qua dự án này, tôi thể hiện:**

- Khả năng xây dựng giao diện web bằng HTML, CSS, JavaScript

- Tư duy phân tích hệ thống (TPS, Chuỗi giá trị – Michael Porter)

- Thiết kế và mô phỏng quy trình nghiệp vụ

- Xử lý và lưu trữ dữ liệu phía client bằng localStorage

- Tư duy chuẩn bị mở rộng sang backend (Node.js / PHP)

## 🎯 Mục tiêu dự án

- Xây dựng website đặt thuê xe hoàn chỉnh

- Mô phỏng hệ thống xử lý giao dịch trong doanh nghiệp

- Lưu trữ và quản lý lịch sử thuê

- Áp dụng lý thuyết MIS vào thực tế

- Chuẩn bị nền tảng để mở rộng sang backend thực tế

## 🛠️ Công nghệ sử dụng

- HTML5 – Xây dựng cấu trúc trang

- CSS3 / Bootstrap – Thiết kế giao diện

- JavaScript (Vanilla JS) – Xử lý logic và nghiệp vụ

- LocalStorage – Lưu trữ dữ liệu phía client

## 💻 Các chức năng chính
**👤 Phía khách hàng (TPS)**

- Xem danh sách xe

- Xem chi tiết xe

- Thêm vào giỏ hàng

- Đặt thuê xe

- Tính tổng tiền tự động theo số ngày thuê

- Lưu lịch sử giao dịch

- Xem lại lịch sử thuê

**⚙️ Xử lý hệ thống**

- Gom nhóm xe trùng

- Tính tổng số lượng và tổng tiền

- Lưu dữ liệu giao dịch vào localStorage

- Mô phỏng quy trình xử lý giao dịch thực tế

## 🧠 Loại hệ thống

- Dự án thuộc loại Hệ thống xử lý giao dịch (TPS) vì:

- Xử lý các giao dịch phát sinh hằng ngày

- Lưu trữ dữ liệu nghiệp vụ

- Đảm bảo tính chính xác của giao dịch

- Là nền tảng cho MIS hoặc DSS trong tương lai

## 🚀 Định hướng phát triển

- Xây dựng backend bằng Node.js hoặc PHP

- Kết nối cơ sở dữ liệu (MySQL / MongoDB)

- Xây dựng hệ thống đăng nhập và phân quyền

- Thêm trang quản trị (Admin Dashboard)

- Triển khai lên hosting
